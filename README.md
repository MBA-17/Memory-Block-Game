# 🧠 Memory Block Game

## 🚀 Overview

This is a simple and interactive **Memory Block Game** built using **HTML5, CSS3, and JavaScript**.
The game challenges users to match pairs of blocks by remembering their positions.

It’s designed as a beginner-friendly frontend project and can be easily reused or customized.

---

## 🎮 How It Works

* You click on blocks to reveal them
* Match two identical blocks to score
* If they don’t match, they flip back
* Complete all matches to win the game

---

## 🛠 Tech Stack

* HTML5 (Structure)
* CSS3 (Styling & Animations)
* JavaScript (Game Logic)

---

## 📦 How to Run the Project

1. Clone or fork this repository:

   ```bash
   git clone https://github.com/MBA-17/Memory-Block-Game.git
   ```

2. Open the project folder

3. Run the game:

   * Simply open `index.html` in your browser
